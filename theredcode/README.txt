# theredcode

**Python · Flask · SQLite · HTML · CSS · Werkzeug**

Веб-приложение-тренажёр для решения задач по Python. Пользователь регистрируется, выбирает главу, открывает задание и отправляет код решения. Система автоматически проверяет функцию по тестам и сохраняет результат в профиле.

Проект выполнен в красно-чёрном минималистичном стиле и подходит для учебной платформы по программированию.

В разработке.

## Возможности

* Регистрация и авторизация пользователей
* Разделение ролей: студент и администратор
* Просмотр глав и заданий
* Отправка Python-кода прямо в браузере
* Автоматическая проверка решений по тестам
* Сохранение истории попыток пользователя
* Личный профиль с результатами
* Админ-панель для просмотра решений
* Добавление новых глав и заданий через администратора

## Установка

### Предварительные условия

* Python 3.10+
* pip

### Запуск проекта

```bash
git clone https://github.com/yudix10/theredcode.git
cd theredcode
```

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте его:

```bash
# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Запустите приложение:

```bash
python app.py
```

После запуска сайт будет доступен по адресу:

```bash
http://127.0.0.1:5000
```

## Основные страницы

* `/register` — регистрация пользователя
* `/login` — вход в аккаунт
* `/` — главная страница с главами
* `/chapter/<id>` — список заданий выбранной главы
* `/task/<id>` — страница решения задачи
* `/profile` — профиль пользователя и результаты
* `/admin` — админ-панель
* `/admin/chapter/add` — добавление главы
* `/admin/task/add` — добавление задания

## Как работает проверка задач

Администратор создаёт задание, указывая:

* название задачи
* описание
* имя функции
* стартовый код
* тесты для проверки

Пользователь пишет решение в поле ввода. После отправки приложение запускает код, ищет нужную функцию и проверяет её на тестовых данных.

Пример задания:

```python
def square(n):
    return n * n
```

Пример тестов:

```python
[(2, 4), (5, 25)]
```
